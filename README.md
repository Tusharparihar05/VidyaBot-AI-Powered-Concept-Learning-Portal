About the Project
VidyaBot is an AI-powered learning portal for students from Class 9 to BTech CSE.
Students type any question in plain language and instantly receive three outputs in parallel:

 Text + Chart — Rich text explanation with an auto-generated bar chart
 Animation — Creatomate-powered explainer video (5–7 slides)
 Avatar Video — HeyGen/D-ID virtual teacher video with whiteboard visuals
