## 📌 Description

VidyaBot is an AI-powered learning portal for students from Class 9 through BTech CSE. Students type any question or concept in plain language and instantly receive three parallel outputs — a structured text explanation with a chart, an animated explainer video, and an avatar-based virtual teacher video — all generated simultaneously using the Claude API, Creatomate, and HeyGen/D-ID.
